VS Code 报错及解决

报错1：无法加载文件 C:\Users\XXXX\AppData\Roaming\npm\yarn.ps1，因为在此系统上禁止运行脚本

解决办法

管理员运行 powershell 并执行命令：set-ExecutionPolicy RemoteSigned

Reference

  • 无法加载文件

报错2：yarn start 命令 无法被识别并启动

解决办法

  1. 退出 VS Code
  2. 右键查看 VS Code 的 属性
  3. 在 兼容性 Tab 页中，勾选 以管理员身份运行此程序
  4. 重新启动 VS Code
