运行环境

XShell

安装

[root@.... ~]# yum install  lrzsz -y

检查安装结果

[root@.... ~]# rpm -qa |grep lrzsz

上传文件（上传到 Linux 服务器）

[root@.... ~]# rz

下载文件（下载到 Windows 客户端）

[root@.... ~]# sz [filename]
